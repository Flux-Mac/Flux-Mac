## 📄 Overview

<a href="#">Flux</a> addresses the mismatch between how evolution shaped human light sensitivity and how modern screen-based work is actually performed. The human circadian system responds to blue-spectrum light as a signal that it is daytime and that sleep should be suppressed — a useful response when blue light came only from the sky, and a problematic one when it comes from a laptop screen at 11pm.

The <a href="#">Backwards Alarm Clock feature</a> counts down to the bedtime Flux calculates as appropriate based on your wake time and sleep duration goals, providing a passive reminder of how much evening time remains before sleep preparation should begin. <a href="#">Notification and app exception system</a> excludes specific applications from color adjustment — color-critical creative software gets true display colors while everything else receives the warmth adjustment — without requiring full manual override.

<a href="#">Multi-display support</a> applies settings independently or consistently across all connected monitors, including external displays and projectors, maintaining the same color temperature management across every screen in use. <a href="#">Keyboard shortcut controls</a> allow rapid temporary disabling of color adjustment, intensity changes, and mode switching without opening the preferences panel for quick access during mode transitions.

The <a href="#">sleep data integration</a> on supported configurations connects Flux's color temperature management to actual measured sleep quality data, enabling fine-tuning of timing and intensity based on how different settings affect sleep outcome rather than relying on general recommendations. <a href="#">Completely free</a> with no premium features, no subscription, and no feature limitations — Flux is supported by the developers as a mission-driven public health tool rather than a commercial product.

## 📌 FAQ

<details>
<summary>What is the Backwards Alarm Clock feature?</summary>

The Backwards Alarm Clock displays a countdown to the bedtime Flux calculates as appropriate for your configured wake time and sleep duration goal, providing a passive reminder of remaining evening time without requiring active attention.

</details>

<details>
<summary>Can Flux manage multiple connected displays differently?</summary>

Yes. Flux can apply settings consistently across all connected displays or configure them independently. External monitors, projectors, and the built-in display can each be managed according to their use.

</details>

<details>
<summary>Does the app exception system work for all applications?</summary>

The exception system excludes specified applications from color adjustment while Flux continues adjusting other applications. This allows color-critical software to display accurate colors without disabling Flux entirely.

</details>

<details>
<summary>How is Flux different from macOS Night Shift?</summary>

Flux offers more granular control over timing, temperature range, and transition speed than Night Shift. It also includes additional modes like Darkroom, Movie mode, and the app exception system. Both accomplish similar goals with different feature depth.

</details>
